Applying the Techniques to other Tabular Problems with Rossmann

Rossmann operates over 3,000 drug stores in 7 European countries. Historical sales data for 1,115 Rossmann stores are provided. The task is to forecast the “Sales” column for the test set. Kaggle hosted it as a competition. These notebooks show how to use NVTabular for doing sales prediction on tabular data.
