WorkflowNode

class nvtabular.workflow.node.WorkflowNode(selector=None)[source]

A WorkflowNode is a group of columns that you want to apply the same transformations to. WorkflowNode’s can be transformed by shifting operators on to them, which returns a new WorkflowNode with the transformations applied. This lets you define a graph of operations that makes up your workflow
